React-Redux Image Browsing App

Released: December 2017

An Imgur browsing site that includes user pages of favorite images. This site utilized the Imgur API, Ruby on Rails, and the Faraday gem on the back-end. Client-side routing used reactrouter and RESTful principles. Redux Thunk middleware also responds to and modifies state changes.

Rails Recipe CMS

Released: November 2017

A recipe sharing community catering to various cuisines and time constraints. This CMS leverages jQuery and JavaScript for AJAX functionality. The development build utilizes the SQLite database schema and configured ActiveRecord associations. The production build on Heroku relies on PostgreSQL. OmniAuth for OAuth authentication with Facebook can currently be setup on the development build.
